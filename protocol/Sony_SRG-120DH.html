Sony SRG-120DH Protocol Info

This document contains information about the VISCA Protocol for the following camera:

  • SRG-120DH

1/2.8″ Exmor CMOS, 12× optical zoom, HDMI/DVI output. Vendor ID: 0001 (Sony), Model ID: 0511.

VISCA Command List

Part 1: Camera-Issued Messages

Function Packet Notes Comments
ACK y0 4z FF z: Socket Number Returned when the command is accepted.
Completion (commands) y0 5z FF z: Socket Number Returned when the command has been executed.
Error – Message Length Error y0 6z 01 FF z: Socket Number Returned when the message length is invalid.
Error – Syntax Error y0 60 02 FF Returned when the command format is different or when a command with an illegal command parameter value is accepted.
Error – Command Buffer Full y0 60 03 FF Returned when a command is received when two command buffers are both in use.
Error – Command Canceled y0 6z 04 FF z: Socket Number Returned when a command in a buffer is canceled by the CANCEL command.
Error – No Socket y0 6z 05 FF z: Socket Number Returned when no command exists in the specified socket.
Error – Command Not Executable y0 6z 41 FF z: Socket Number Returned when the command cannot be executed due to current conditions.
Network Change y0 38 FF Returned when power is supplied to the camera. Not available for VISCA over IP.

Block Inquiries

Byte numbering is absolute: byte 0 is the address byte (y0), byte 1 is 50, data begins at byte 2, and the final byte is the FF terminator. Bit 0 is the least significant bit. The :: operator denotes concatenation, with the left operand as the most significant bits.

Pan/Tilt Status Codes

Response from Pan-tiltModeInq (8x 09 06 10 FF) returns y0 50 pq rs FF where pqrs is a 16-bit status field:

Bits Status
---- ---- 0--- ---1 Panning reaches left limit
---- ---- 0--- --1- Panning reaches right limit
---- ---- 0--- -1-- Tilting reaches upper limit
---- ---- 0--- 1--- Tilting reaches lower limit
---- ---- 0-00 ---- Pan functions normally
---- ---- 0-01 ---- Pan/tilt position cannot be detected
---- ---- 0-10 ---- Pan mechanism is defective
---- --00 0--- ---- Tilt functions normally
---- --01 0--- ---- Tilt position cannot be detected
---- --10 0--- ---- Tilt mechanism is defective
---- 00-- 0--- ---- No movement instruction
---- 01-- 0--- ---- Pan/Tilt operating
---- 10-- 0--- ---- Pan/Tilt operations complete
---- 11-- 0--- ---- Pan/Tilt operating failed
--00 ---- 0--- ---- Not initialized
--01 ---- 0--- ---- Initializing
--10 ---- 0--- ---- Initialization complete
--11 ---- 0--- ---- Initialization failed

( – : optional )
