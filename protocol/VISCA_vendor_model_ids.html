VISCA Vendor and Model IDs

This document lists known Vendor IDs and Model IDs returned by the CAM_VersionInq command across VISCA-compatible cameras.

CAM_VersionInq Response Formats

Different manufacturers implement the CAM_VersionInq response differently.

Sony Format

Field Bytes Description
Inquiry Packet 8x 09 00 02 FF
Response y0 50 GG GG HH HH JJ JJ KK FF
GGGG Vendor ID
HHHH Model ID
JJJJ ROM Revision
KK Maximum Socket Number (typically 02)

PTZOptics Format

Field Bytes Description
Inquiry Packet 81 09 00 02 FF
Response 90 50 ab cd mn pq rs tu vw FF
ab Factory Code (00=VHD, 01=MR, 08=T)
cd Hardware Version
mnpq ARM Version
rstu FPGA Version
vw Camera Model (01=C Type, 02=M Type, 03=S Type)
