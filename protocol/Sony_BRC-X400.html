Sony BRC-X400 Protocol Info

This document contains information about the VISCA Protocol for the following cameras:

  • BRC-X400
  • BRC-X401
  • SRG-X400
  • SRG-X402
  • SRG-201M2
  • SRG-X120
  • SRG-HD1M2

Note: Software Version 2.10

VISCA Command List

Part 1: Camera-Issued Messages

Function Packet Notes Comments
ACK y0 4z FF z: Socket Number Returned when the command is accepted.
Completion (commands) y0 5z FF z: Socket Number Returned when the command has been executed.
Error – Syntax Error y0 60 02 FF Returned when the command format is different or when a command with an illegal command parameter value is accepted.
Error – Command Buffer Full y0 60 03 FF Returned when a command is received when two command buffers are both in use.
Error – Command Canceled y0 6z 04 FF z: Socket Number Returned when a command in a buffer is canceled by the CANCEL command.
Error – No Socket y0 6z 05 FF Returned when no command exists in the specified socket.
Error – Command Not Executable y0 6z 41 FF Returned when the command cannot be executed due to current conditions.
Network Change y0 38 FF Returned when the network configuration changes.
