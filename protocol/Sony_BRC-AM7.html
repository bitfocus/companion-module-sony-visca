Sony BRC-AM7 Protocol Info

This document contains information about the VISCA Protocol for the following camera:

  • BRC-AM7

Note: Software Version 2.0

VISCA Command List

Part 1: Camera-Issued Messages

Function Packet Notes Comments
ACK y0 4z FF z: Socket Number Returned when the command is accepted.
Completion (commands) y0 5z FF z: Socket Number Returned when the command has been executed.
Completion (inquiries) y0 5z ... FF z: Socket Number Returned when the inquiry has been executed. Reply data follows byte 3.
Error – Message Length Error y0 6z 01 FF z: Socket Number Returned when the message length is incorrect.
Error – Syntax Error y0 6z 02 FF z: Socket Number Returned when the command format is different or when a command with illegal command parameters is received.
Error – Command Buffer Full y0 6z 03 FF Could not accept a command that is received while two commands are currently being executed (two sockets have been used).
Error – Command Canceled y0 6z 04 FF z: Socket Number Returned when a command which is being executed in a socket specified by the cancel command is canceled. The completion message for the command is not returned.
Error – No Socket y0 6z 05 FF z: Socket Number Returned when no command is executed in a socket specified by the cancel command, or when an invalid socket number is specified.
Error – Command Not Executable y0 6z 41 FF z: Socket Number Returned when a command cannot be executed due to current conditions.
Network Change*1 y0 38 FF Issued when power is supplied to the camera.

*1 Not available for VISCA over IP.

Note: Press/Release commands imitate a button press and release. When a Press command is received, the operation starts. When a Release command is received, the operation stops. Note: The tally lamp is turned off if not receiving an On command from any controller for 15 seconds after receiving an On command (VISCA over IP).

Press/Release Commands

The following commands use press/release patterns. When a Press command is received, the operation starts. When a Release command is received, the operation stops. For long-press operations, maintain at least 1 second between Press and Release.

Multi Function Dial Command

Controls the GUI cursor operation. CW/CCW moves the GUI cursor, and Press/Release applies the setting at the cursor position. If a Press/Release pair is received on the shooting screen, direct menu operation starts. Face/eye detection frame selection and focus area movement are not supported via these commands.

Direct Menu Command

Moves the cursor to a specified setup item in the direct menu. Can also be used to exit the direct menu. The movement destination depends on the interval between Press and Release:
