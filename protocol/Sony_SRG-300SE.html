Sony SRG-300SE / SRG-301SE / SRG-201SE — VISCA Protocol Reference

Source: Sony SRG-300SE Technical Manual (C-092-100-12(1)). HD Color Video Camera. RS-422 interface (9600/38400 bps). SRG-300SE/301SE: 30× optical zoom. SRG-201SE: 20× optical zoom.

Pan range: ±170°. Tilt range: −20° to +90° (Image Flip Off) or −90° to +20° (Image Flip On). 16 presets (0–F).

Part 1: Camera-Issued Messages

Acknowledge / Completion

Message Packet Comments
Acknowledge z0 4y FF y: Socket number (1 or 2)
Completion z0 5y FF y: Socket number

z = camera address + 8.

Error Messages

Error Packet Comments
Syntax Error z0 60 02 FF Unsupported command or missing parameters
Command Buffer Full z0 60 03 FF Two commands already executing
Command Canceled z0 6y 04 FF y: Socket number of canceled command
No Socket z0 6y 05 FF No command in specified socket, or invalid socket
Command Not Executable z0 6y 41 FF Cannot execute in current mode

Network Change

Message Packet Comments
Network Change z0 38 FF Issued when power is supplied to the camera

Max pan speed: 0x18. Max tilt speed: 0x17. Values up to 0x7F are accepted but operate at maximum speed.

Pan/Tilt Speed (Slow Mode On)

When Slow Pan/Tilt mode is enabled, speed values 01–7F are available with fine-grained control. Speeds 01–18 provide the same speeds as Slow Mode Off at matching parameters. Speeds 19–7F provide sub-degree granularity from 0.10–11.18°/s (pan and tilt identical).
