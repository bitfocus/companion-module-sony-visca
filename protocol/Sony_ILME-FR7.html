Sony ILME-FR7 Protocol Info

This document contains information about the VISCA Protocol for the following cameras:

  • ILME-FR7
  • ILME-FR7K

Note: Software Version 4.00

VISCA Command List

Part 1: Camera-Issued Messages

Function Packet Notes Comments
ACK y0 4z FF z: Socket Number Returned when the command is accepted.
Completion (commands) y0 5z FF z: Socket Number Returned when the command has been executed.
Completion (inquiries) y0 5z ... FF z: Socket Number Returned when the inquiry has been executed. Reply data follows byte 3.
Error – Message Length Error y0 6z 01 FF z: Socket Number Returned when the message length is incorrect.
Error – Syntax Error y0 6z 02 FF z: Socket Number Returned when the command format is different or when a command with illegal command parameters is received.
Error – Command Buffer Full y0 6z 03 FF Could not accept a command that is received while two commands are currently being executed (two sockets have been used).
Error – Command Canceled y0 6z 04 FF z: Socket Number Returned when a command which is being executed in a socket specified by the cancel command is canceled.
Error – No Socket y0 6z 05 FF z: Socket Number Returned when no command is executed in a socket specified by the cancel command, or when an invalid socket number is specified.
Error – Command Not Executable y0 6z 41 FF z: Socket Number Returned when a command cannot be executed due to current conditions.

Notable Differences from BRC-X400

  • 20-bit pan/tilt coordinates – The FR7 uses 5-nibble (20-bit) pan/tilt position values (e.g., 0p 0p 0p 0p 0p) versus the BRC-X400’s 4-nibble (16-bit) values. This provides finer positioning resolution across the FR7’s ±170° pan and −30°/+195° tilt range.
  • Extended speed step mode – In addition to Normal mode (24 steps, 01–18), the FR7 supports an Extended mode with 50 speed steps (01–32), selectable via 8x 01 06 45 pp FF (pp: 08=Normal, 18=Extended). The BRC-X400 has a fixed 24-step range.
  • Cinema-oriented ND filter system – The FR7 has a built-in variable ND filter (1/4–1/128 in 21 steps), preset ND positions (Clear, Preset 1–3), and auto ND control. These use the 7E 04 extended command space. The BRC-X400 has no ND filter commands.
  • Press/release command pattern – Many FR7 controls use press/release pairs that emulate physical button behavior, including Recording, Display, Assignable Buttons, Multi Selector, Multi Function Dial, Push AF/MF, and Direct Menu. Long-press operations require holding the Press state for 1 second or longer before sending Release.
  • No traditional exposure direct commands – The FR7 does not use the standard Sony CAM_AE mode command (04 39) or the standard iris/shutter/gain direct commands (04 4B, 04 4A, 04 4C). Instead, it provides Auto Iris (05 34), AGC (7E 01 75), and Auto Shutter (05 35) on/off toggles, with iris adjusted via up/down step commands in the 7E 04 4B extended space.
  • Audio level control – The FR7 includes VISCA commands for audio input level control (Master, CH1, CH2) with auto/manual mode switching and direct level setting (0–99). The BRC-X400 has no audio commands.
  • Shooting mode inquiry – The FR7 supports a shooting mode inquiry (05 30) returning Custom, Cine EI, Flexible ISO, or Cine EI Quick modes. This is specific to the FR7’s cinema camera functionality.
  • Knee permission inquiry – The FR7 adds a knee permission inquiry (7E 04 6D) that reports whether each knee parameter is enabled, disabled, or display-only, reflecting the FR7’s context-dependent parameter availability.
  • PTZ Auto Framing – The FR7 includes a PTZ Auto Framing on/off command (7E 04 3A) for automatic subject tracking.
  • Separate preset duration mode – In addition to speed-based preset recall, the FR7 supports duration-based preset recall where each preset can specify a time (1–99 seconds) for the movement to complete.
