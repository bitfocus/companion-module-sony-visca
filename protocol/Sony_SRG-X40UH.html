Sony SRG-X40UH Protocol Info

This document contains information about the VISCA Protocol for the following cameras:

  • SRG-X40UH
  • SRG-H40UH

Note: Software Version 1.00

SRG-X40UH supports 4K (2160p) output; SRG-H40UH does not. Features available only on the SRG-X40UH are marked with †.

VISCA Command List

Part 1: Camera-Issued Messages

Function Packet Notes Comments
ACK y0 4z FF z: Socket Number Returned when the command is accepted.
Completion (commands) y0 5z FF z: Socket Number Returned when the command has been executed.
Error – Syntax Error y0 60 02 FF Returned when the command format is different or when a command with an illegal command parameter value is accepted.
Error – Command Buffer Full y0 60 03 FF Returned when a command is received when two command buffers are both in use.
Error – Command Canceled y0 6z 04 FF z: Socket Number Returned when a command in a buffer is canceled by the CANCEL command.
Error – No Socket y0 6z 05 FF z: Socket Number Returned when no command exists in the specified socket.
Error – Command Not Executable y0 6z 41 FF z: Socket Number Returned when the command cannot be executed due to current conditions.
Network Change y0 38 FF Issued when power is supplied to the camera.

Pan/Tilt Status Code

Bits Status
pppp & 0x0001 Panning reaches left end
pppp & 0x0002 Panning reaches right end
pppp & 0x0004 Tilting reaches upper limit
pppp & 0x0008 Tilting reaches lower limit
pppp[7:6] = 00 Pan functions normally
pppp[7:6] = 01 Pan mechanism defective
pppp[9:8] = 00 Tilt functions normally
pppp[9:8] = 01 Tilt mechanism defective
pppp[11:10] = 00 No movement instruction
pppp[11:10] = 01 Pan/Tilt operating
pppp[11:10] = 10 Pan/Tilt operations complete
pppp[11:10] = 11 Pan/Tilt operating failed
pppp[13:12] = 00 Not initialized
pppp[13:12] = 01 Initializing
pppp[13:12] = 10 Initialization complete
pppp[13:12] = 11 Initialization failed
